# Kemi UI Monorepo

A monorepo containing Kemi UI web components library and its Vue wrapper.

## Packages

- **[@klpod221/kemi-ui](./packages/core)** - Core web components library built with Lit
- **[@klpod221/kemi-ui-vue](./packages/vue)** - Vue 3 wrapper for Kemi UI

## Development

```bash
# Install dependencies
npm install

# Build all packages
npm run build

# Build specific package
npm run build:core
npm run build:vue

# Dev mode for specific package
npm run dev:core
npm run dev:vue
```
